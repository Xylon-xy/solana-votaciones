Blockchain Debate es una dApp (Aplicación Descentralizada) diseñada para el Hackathon, que permite crear debates en tiempo real y registrar votos de manera transparente e inmutable.

 Características Principales
- Interfaz Web3 Pro: Diseño oscuro (Dark Mode) optimizado para la mejor experiencia de usuario (UX).
- Votación Híbrida: Soporte para conexión directa con Phantom Wallet y modo Invitado para pruebas rápidas.
- Duelo en Tiempo Real: Barra de choque dinámica que muestra la tendencia del debate al instante.
- Feedback Visual: Sistema de partículas (confeti) diferenciado por bando (Verde para SÍ, Rojo para NO).
- Historial Persistente: Gestión de múltiples debates guardados en el almacenamiento local del navegador.

 Stack Tecnológico
- Frontend: HTML5, CSS3 (Animaciones personalizadas) y JavaScript (Vanilla JS).
- Web3 Engine: @solana/web3.js para la interacción con el cluster de Solana.
- Efectos: Confetti-js para la gamificación de la votación.

Estructura del Proyecto
- index.html: Estructura de la dApp y estilos integrados para máxima velocidad de carga.
- app.js: Lógica de negocio, gestión de estados de votación y conexión con la red de Solana.

Instalación y Uso
1. Clona este repositorio.
2. Abre index.html en cualquier navegador moderno.
3. ¡Crea un tema, conecta tu wallet y empieza a debatir!
